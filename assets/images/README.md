# Logo Images

Place your campus logo images here:

- `uw-logo.png` (or `.jpg`, `.svg`) - University of Washington logo
- `umd-logo.png` (or `.jpg`, `.svg`) - University of Maryland logo

## Recommended Image Specifications:
- Format: PNG (with transparency) or JPG
- Size: 200-400px wide (height will scale proportionally)
- Aspect Ratio: Horizontal/landscape orientation works best

## How to Use:

1. **Add your images here:**
   - Copy `uw-logo.png` to this folder
   - Copy `umd-logo.png` to this folder

2. **The code will automatically use them** once you update the logo paths in `src/App.jsx`
